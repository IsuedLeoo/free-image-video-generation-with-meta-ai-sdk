Introduction

The metaai_api package is an unofficial Python SDK for Meta AI. It wraps Meta's internal GraphQL and SSE APIs to provide a clean Python interface for:

Image Generation
Text-to-image generation with orientation control (16:9, 9:16, 1:1)
Video Generation
Text-to-video with auto-polling and async job support
Image Analysis
Upload images and ask Llama 3 to describe or analyze them
Chat & Reasoning
Full conversational AI with streaming support
How it works
The SDK authenticates using browser cookies from meta.ai — the same session your browser uses. It communicates with Meta's internal DGW (Dynamic Gateway) via GraphQL SSE subscriptions, the same protocol the web interface uses.

Installation

bash 
pip install metaai-api

Dependencies

The package automatically installs these dependencies:

  • requests — HTTP client for API calls
  • requests-html — HTML rendering for page scraping
  • python-dotenv — Environment variable loading
  • beautifulsoup4 — HTML parsing for challenge pages

Optional Dependencies

For the HTTP REST server mode:

bash 
pip install metaai-api[server]

This installs fastapi, uvicorn, and pydantic.

Authentication

Meta AI uses cookie-based authentication — no API keys or bearer tokens needed. The SDK authenticates as if it were your browser session.

Required Cookies

Cookie Purpose Lifetime Required
datr Device identifier ~2 years Required
ecto_1_sess Session token ~1 hour Required
abra_sess Extended session ~30 days Optional

Getting Your Cookies

  1. Open https://meta.ai in Chrome and log in
  2. Open DevTools: F12 or Cmd + Option + I
  3. Navigate to Application tab → Cookies → https://meta.ai
  4. Copy the values for datr and ecto_1_sess

Configuring the SDK

Create a .env.metaai file in your project root:

env 
META_AI_DATR=your_datr_value
META_AI_ECTO_1_SESS=your_ecto_1_sess_value
META_AI_ABRA_SESS=your_abra_sess_value

Or pass cookies directly in code:

python 
from metaai_api import MetaAI

ai = MetaAI(cookies={
    "datr": "your_datr_value",
    "ecto_1_sess": "your_ecto_1_sess_value"
})
Cookie Expiry
The ecto_1_sess cookie expires frequently (~1 hour). If API calls fail with authentication errors, refresh this cookie from your browser. The SDK will auto-extract a fresh access token from the meta.ai page on initialization.

Image Generation

Generate images from text prompts using Meta's image generation model. Generation typically takes 30-120 seconds.

Method Signature

python 
ai.generate_image_new(
    prompt: str,
    orientation: str = "LANDSCAPE",
    num_images: int = 1
) -> dict

Parameters

Parameter Type Default Description
prompt str — Text description of the image to generate
orientation str "LANDSCAPE" Aspect ratio: "LANDSCAPE" (16:9), "VERTICAL" (9:16), or "SQUARE" (1:1)
num_images int 1 Number of images to generate (max 4)

Response Schema

json 
{
  "success": true,           // Whether generation succeeded
  "image_urls": [             // List of generated image URLs
    "https://scontent.fyvr4-1.fna.fbcdn.net/v/t39.8562-6/..."
  ],
  "status": "READY",         // READY, PROCESSING, or FAILED
  "error": null              // Error message if failed
}

Example: Multiple Orientations

python 
from metaai_api import MetaAI

ai = MetaAI()

for orientation in ["LANDSCAPE", "VERTICAL", "SQUARE"]:
    result = ai.generate_image_new(
        prompt="a serene mountain landscape at golden hour",
        orientation=orientation,
        num_images=1
    )
    if result["success"]:
        print(f"{orientation}: {result['image_urls'][0]}")

Video Generation

Generate short video clips from text prompts. Videos are generated asynchronously — the SDK polls for completion automatically.

Method Signature

python 
ai.generate_video_new(
    prompt: str,
    auto_poll: bool = True,
    max_poll_attempts: int = 15,
    poll_wait_seconds: int = 3
) -> dict

Parameters

Parameter Type Default Description
prompt str — Text description of the video to generate
auto_poll bool True Wait for video to be ready before returning
max_poll_attempts int 15 Maximum number of polling attempts
poll_wait_seconds int 3 Seconds between polling attempts

Response Schema

json 
{
  "success": true,
  "video_urls": [           // Direct video file URLs
    "https://video.fyvr4-1.fna.fbcdn.net/v/t42.1792-2/..."
  ],
  "media_ids": ["1234567890"],  // Meta internal media IDs
  "conversation_id": "abc123",
  "status": "READY"
}

Example: Generate and Download

python 
import requests
from metaai_api import MetaAI

ai = MetaAI()

result = ai.generate_video_new(
    prompt="timelapse of stars moving across the night sky",
    auto_poll=True,
    max_poll_attempts=20,
    poll_wait_seconds=4
)

if result["success"]:
    url = result["video_urls"][0]
    response = requests.get(url)
    with open("stars_timelapse.mp4", "wb") as f:
        f.write(response.content)
    print(f"Saved video: {len(response.content)} bytes")
Generation Time
Video generation typically takes 20-45 seconds. The SDK polls automatically with adaptive backoff. For longer videos, increase max_poll_attempts or poll_wait_seconds.

Image Analysis (Vision)

Upload an image and ask Meta AI to describe, analyze, or answer questions about it. This uses Llama 3's vision capabilities.

Workflow

  1. Upload the image to get a media_id
  2. Prompt with your question, passing the media_id

Example

python 
from metaai_api import MetaAI

ai = MetaAI()

# Step 1: Upload the image
upload = ai.upload_image("/path/to/photo.jpg")
print(f"Upload success: {upload['success']}")

if upload["success"]:
    media_id = upload["media_id"]

    # Step 2: Ask about the image
    response = ai.prompt(
        "What objects are in this image? Describe the scene.",
        media_ids=[media_id]
    )
    print(response["message"])

Supported Image Formats

JPEG, PNG, WebP, GIF (non-animated). Maximum file size: 10MB.

Image Upload

Upload images to Meta AI's servers for use in analysis or as generation references.

Method Signature

python 
ai.upload_image(file_path: str) -> dict

Response Schema

json 
{
  "success": true,
  "media_id": "1234567890abcdef",  // Use this in prompt() calls
  "file_size": 245862
}
Upload Reliability
Uploads use Meta's rupload.meta.ai endpoint with OAuth authentication. The SDK handles 412 (Precondition Failed) errors with exponential backoff automatically.

Streaming Responses

Enable streaming to receive tokens as they're generated, instead of waiting for the complete response.

python 
from metaai_api import MetaAI

ai = MetaAI()

# Stream tokens as they arrive
for chunk in ai.prompt("Write a poem about artificial intelligence", stream=True):
    print(chunk, end="", flush=True)
How Streaming Works
The SDK uses Server-Sent Events (SSE) over Meta's GraphQL streaming endpoint. Tokens are yielded as they arrive from the model, typically at ~50ms intervals.

Async Video Jobs

For applications that need to submit multiple videos without blocking, use the async pattern:

python 
from metaai_api import MetaAI
import time

ai = MetaAI()

# Submit without waiting
job = ai.generate_video_new(
    prompt="a cat playing piano",
    auto_poll=False
)
print(f"Job submitted: {job['conversation_id']}")

# Poll manually
while True:
    time.sleep(5)
    result = ai.generate_video_new(
        prompt="a cat playing piano",
        auto_poll=True
    )
    if result["status"] == "READY":
        break

print(f"Video ready: {result['video_urls']}")

Extending Videos

Extend an existing generated video with additional content using its media ID.

python 
from metaai_api import MetaAI

ai = MetaAI()

result = ai.extend_video(
    source_media_id="1234567890",  # From a previous generation
    prompt="continue the scene with rain starting to fall"
)
print(result)

Error Handling

The SDK provides structured error responses and custom exceptions.

Exception Types

Exception When Raised
FacebookInvalidCredentialsException Cookie authentication fails or session expires
FacebookRegionBlocked Meta AI is not available in your region

Error Response Pattern

python 
from metaai_api import MetaAI, FacebookInvalidCredentialsException

try:
    ai = MetaAI()
    result = ai.generate_image_new(prompt="test")
except FacebookInvalidCredentialsException:
    print("Session expired. Please refresh your cookies.")
except FacebookRegionBlocked:
    print("Meta AI is not available in your region.")

Retry Logic

The SDK automatically retries failed prompt() calls up to 3 times with exponential backoff. GraphQL errors like GRAPHQL_VALIDATION_FAILED trigger automatic fallback to alternative doc IDs.

Challenge Pages

Meta AI occasionally presents bot-detection challenge pages. The SDK handles these automatically:

  1. Detects challenge pages via HTML content analysis
  2. Submits verification POST requests to clear the challenge
  3. Extracts fresh cookies and access tokens from the response
  4. Retries the original request with updated credentials
Automatic Handling
In most cases, you don't need to worry about challenge pages. The SDK's detect_challenge_page() and handle_meta_ai_challenge() utilities handle the full flow transparently.

API Reference

Complete reference for all public methods on the MetaAI class.

MetaAI Class

python 
class MetaAI:
    def __init__(
        self,
        cookies: dict = None,
        env_path: str = ".env.metaai"
    )
prompt (message, stream=False, media_ids=None)
Send a text message to Meta AI. Returns a dict with message and sources. When stream=True, returns a generator yielding text chunks.
generate_image_new (prompt, orientation="LANDSCAPE", num_images=1)
Generate images from a text prompt. Returns a dict with image_urls, status, and success.
generate_video_new (prompt, auto_poll=True, max_poll_attempts=15, poll_wait_seconds=3)
Generate a video from a text prompt. When auto_poll=True, blocks until the video is ready. Returns a dict with video_urls, media_ids, and status.
upload_image (file_path)
Upload an image file to Meta AI servers. Returns a dict with media_id for use in prompt() calls.
extend_video (source_media_id, prompt)
Extend an existing video with new content. The source_media_id comes from a previous generate_video_new() call.
retry (func, *args, **kwargs)
Retry wrapper with exponential backoff. Used internally for prompt() calls.

HTTP REST Server

The SDK includes a FastAPI server for exposing Meta AI capabilities as a REST API. Useful for microservices or integrating with non-Python applications.

Starting the Server

bash 
python -m metaai_api.api_server

Server runs on http://localhost:8000 by default.

Endpoints

Method Endpoint Description
POST /chat Send a text message to Meta AI
POST /image Generate images (with timeout protection)
POST /video Generate videos (with auto-polling)
POST /video/async Submit async video generation job
GET /video/jobs/{job_id} Check async job status
POST /video/extend Extend an existing video
POST /upload Upload an image file
GET /healthz Health check endpoint

Example: Chat Endpoint

bash 
curl -X POST http://localhost:8000/chat \
  -H "Content-Type: application/json" \
  -d '{"message": "What is the capital of France?"}'

Example: Image Generation Endpoint

bash 
curl -X POST http://localhost:8000/image \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "a beautiful sunset over mountains",
    "orientation": "LANDSCAPE",
    "num_images": 1
  }'

CLI Reference

The skill includes a CLI wrapper for quick terminal usage without writing Python code.

Commands

bash 
# Generate an image
python metaai_generate.py image "a beautiful sunset" --orientation LANDSCAPE --num 2

# Generate a video
python metaai_generate.py video "waves on a beach" --no-poll

# Analyze an image
python metaai_generate.py analyze photo.jpg --question "What colors dominate?"

# Chat
python metaai_generate.py chat "Tell me a joke"

# Check cookie status
python metaai_generate.py check

Options

Command Option Description
image --orientation LANDSCAPE, VERTICAL, or SQUARE
image --num Number of images (1-4)
video --no-poll Skip waiting for video completion
analyze --question Custom analysis prompt

Examples

Batch Image Generation

python 
from metaai_api import MetaAI
import json

ai = MetaAI()

prompts = [
    "a red fox in a forest",
    "a robot painting a portrait",
    "an astronaut riding a horse on Mars"
]

results = []
for prompt in prompts:
    result = ai.generate_image_new(prompt=prompt, num_images=1)
    results.append(result)
    print(f"Done: {prompt[:30]}...")

# Save all results
with open("generated_images.json", "w") as f:
    json.dump(results, f, indent=2)

Multi-Modal Conversation

python 
from metaai_api import MetaAI

ai = MetaAI()

# Upload a chart image
upload = ai.upload_image("chart.png")
media_id = upload["media_id"]

# Ask about the chart
response = ai.prompt(
    "What trends does this chart show? Summarize the key insights.",
    media_ids=[media_id]
)
print(response["message"])

# Follow up without re-uploading
response2 = ai.prompt(
    "What would the projection be for next quarter based on this data?"
)
print(response2["message"])

Streaming Chat Interface

python 
from metaai_api import MetaAI
import sys

ai = MetaAI()

print("Meta AI Chat (type 'quit' to exit)")
print("=" * 40)

while True:
    user_input = input("\nYou: ")
    if user_input.lower() == "quit":
        break

    print("\nAI: ", end="", flush=True)
    for chunk in ai.prompt(user_input, stream=True):
        print(chunk, end="", flush=True)
    print()
